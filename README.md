# ITPM_Y3_S1
Smart Hostel &amp; Roommate Matching System
The Smart Hostel & Roommate Matching System is a web-based application developed using the MERN stack (MongoDB, Express.js, React.js, and Node.js). This system is designed to digitally transform traditional hostel management by introducing intelligent roommate matching, structured room allocation, and workflow-based complaint handling.
The primary objective of this project is to improve the overall hostel living experience for university students while reducing the administrative workload for wardens and hostel administrators. Instead of random roommate assignments, the system introduces a compatibility-based matching algorithm where students submit their lifestyle preferences such as sleep schedule, cleanliness level, study habits, smoking preference, and noise tolerance. Based on these inputs, the system calculates a compatibility score and suggests suitable roommates.
In addition to roommate matching, the system includes a Room Allocation & Management module that allows administrators to manage room capacity, availability, and approval workflows. This prevents overbooking and ensures transparent room assignment. The Complaint & Maintenance Management module enables students to digitally submit maintenance issues, track complaint status (Pending → In Progress → Resolved), and maintain accountability through structured workflows
